## Install and use

- Get the project code

```bash
git clone https://github.com/helptome1/wallhaven-vue3.git
```

- Installation dependencies

```bash
cd wallhaven-vue3
yarn
```

- run

```bash
yarn electron:start
```

- build

```bash
yarn make
```
